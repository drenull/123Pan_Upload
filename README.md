# 123Pan_Upload
123云盘秒传脚本<br>
自用，所以比较简陋<br>
建议运行在Windows Terminal<br>
秒传的目标文件需要存在于123云盘的存储池<br>
提供目标文件的 MD5、文件名和字节数<br>
文件名可以任意填写<br>

示例：<br>
文件MD5：ade0288de0e4ee83901e0e454ce5681a<br>
文件名：fuxi_images_V14.0.9.0.TMCCNXM_20221211.0000.00_13.0_cn_ade0288de0.tgz<br>
文件字节数：8278402273<br>

MD5 和 字节数 是必须的<br>
